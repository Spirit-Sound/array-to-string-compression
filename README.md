# array-to-string-encoding
Encoding array to string

## Installation

    npm install array-to-string-encoding

## Usage

    var arrayToString = require('array-to-string-encoding');

    arrayToString([1, 2, 3, 5, 6, 8, 10, 12], function(err, result){
        console.log(result);
    })

## Test
